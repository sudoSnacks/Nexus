# Backend - Supabase

This directory contains the database schema and edge functions for the Event Management System.

## Contents
- `schema.sql`: SQL scripts to set up the database.
- `supabase/`: Supabase project configuration and functions.

## Getting Started
1. Install Supabase CLI.
2. Run `supabase start` to start the local development environment.
3. Apply schema: `supabase db reset` (or run sql from `schema.sql`).
